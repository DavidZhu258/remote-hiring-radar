# Scripts: remote-hiring-radar

Recommended first scripts:

1. `validate_gold.py` - parse JSONL and check required fields.
2. `run_seed_eval.py` - run seed cases against the current system.
3. `score_results.py` - compute rubric metrics and write `evals/results/YYYY-MM-DD_seed.json`.

Do not commit secrets, private raw datasets, or model weights.
